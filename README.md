# Russian Stocks Scripts

Набор Python-скриптов для анализа акций российских компаний на Московской бирже.

## 📊 Возможности

- **Расчет финансовых показателей**: P/E, P/B, P/S, дивидендная доходность
- **Автоматическое получение данных**: Цены и количество акций с API Московской биржи
- **Графический анализ**: Построение графиков P/E и цен
- **Рекомендации**: Автоматические рекомендации на основе анализа

## 🚀 Быстрый старт

### Установка зависимостей

```bash
# Убедитесь, что установлен Python 3.7+
python --version

# Установите необходимые библиотеки
pip install matplotlib
```

### Анализ акции

```bash
# Перейдите в папку universal
cd universal

# Анализ Роснефти
python analyze_stock.py ROSN

# Анализ Северстали с целевым P/E=8
python analyze_stock.py CHMF --target-pe 8

# Показать список доступных тикеров
python analyze_stock.py --list
```

## 📈 Использование

### Анализ компании (`analyze_stock.py`)

Скрипт запрашивает:
- Автоматически: текущую цену и количество акций с Московской биржи
- Вручную: финансовые показатели из отчётности компании

**Пример вывода:**
```
======================================================================
  Роснефть (ROSN) — финансовый анализ
  2025-04-11 11:45
======================================================================

Получение данных с Московской биржи...
  Цена акции:         550.50 RUB
  Акции в обращении:  10,750,000,000 шт.

ВВЕДИТЕ ФИНАНСОВЫЕ ПОКАЗАТЕЛИ КОМПАНИИ:
  (данные из отчётности, в млрд RUB)

Чистая прибыль Роснефти:
  Чистая прибыль (млрд RUB): 1200
...
```

## 🛠 Модули

### `moex_api.py`
- Класс `MoexAPI` для работы с API Московской биржи
- Получение текущих цен и количества акций
- Загрузка исторических данных
- Вспомогательные функции для форматирования

### `financial_metrics.py`
- Класс `FinancialMetrics` для расчета показателей
- P/E, P/B, P/S, дивидендная доходность
- Формирование отчетов и интерпретация результатов

## 📊 Поддерживаемые тикеры

- `ROSN` - Роснефть
- `CHMF` - Северсталь
- `GAZP` - Газпром
- `SBER` - Сбербанк
- `LKOH` - Лукойл
- `MGNT` - Магнит
- `NVTK` - Новатэк
- `TATN` - Татнефть
- `ALRS` - АЛРОСА
- `POLY` - Полюс
- `MOEX` - Московская биржа
- `YNDX` - Яндекс
- `OZON` - Ozon
- `TCSG` - TCS Group (Тинькофф)
- `VTBR` - ВТБ

## 🔧 Оригинальные скрипты

Проект также содержит оригинальные скрипты:

- `rosneft_pe.py` - Расчет P/E для Роснефти
- `severstal_pe.py` - Расчет P/E для Северстали

## ⚠️ Важные замечания

1. **Финансовые показатели** необходимо вводить вручную из официальных отчётов компаний
2. **Целевой P/E=7** используется по умолчанию как ориентир для российского рынка
3. **Анализ является упрощённым** и не учитывает все факторы
4. **Консультируйтесь с финансовым советником** перед принятием инвестиционных решений
